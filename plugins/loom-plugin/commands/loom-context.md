---
name: loom-context
description: Print a compressed codebase overview (~200 tokens) — modules, hot functions, summary coverage. Use at session start to orient without reading files.
---

$ARGUMENTS

Load the `loom://primer` MCP resource, or run `loom context` CLI for the same output.

Shows: repo shape (node/edge counts), module breakdown, top entry-point functions, summary coverage percentage, last analyzed timestamp.
